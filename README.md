## Yangrtc Overview
 Yangrtc is a cross-platform webRTC SDK, more suitable for embedded/Internet of Things webRTC applications.
 
## Key Features
+ Audio/Video Support  
H264/H265  
Opus/G711A/G711U/AAC/MP3/SPEEX  
AEC/ANS/AGC
+ DataChannels
+ NACK/PLI/FIR/FEC/TWCC
+ STUN/TURN Support
+ IPv4/IPv6
+ Whip/Whep  
+ Linux/Android/Windows
+ MP4/FLV Recording  
AAC/H264  
AAC/H265(HEVC)
